  •  package Kalah

    The Kalah framework for the AI course at FAU WS16/17.

    The Kalah framework for the AI course at FAU WS16/17. This package contains:

    • The abstract Agent class, which is supposed to be extended by your agents.
    • A HumanPlayer agent you can use to test your agents
    • A info.kwarc.teaching.AI.Kalah.Board class representing a game board
    • A info.kwarc.teaching.AI.Kalah.Game class that lets two agents play each other on a new board.
